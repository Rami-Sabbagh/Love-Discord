http_utils.request(url, data, method, headers, useMultipart)
Execute a http request synchronously.

Parameters:

  • url string or table The request url, or a url table in lua-socket format.
  • data string or table The request body data.
  • method optional string The request method: GET, POST, etc.
  • headers optional table The request headers table.
  • useMultipart optional boolean Whether use multipart/form-data instead of json to encode the request data (when provided in a table).

Returns:

  1. boolean true then the request has been done successfully, false otherwise.
  2. string or table The error string on failure, otherwise it's the reponse body data, automatically decode if it was JSON.
  3. optional table The response headers (only on success).
  4. optional number The response status code (only on success).
  5. optional string The response status line (only on success).
