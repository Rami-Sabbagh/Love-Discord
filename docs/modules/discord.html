Verify (value, name, ...)
A function for verifying the arguments types of a method. If the argument verification fails, then an error with level 3 is raised (the caller level).

Parameters:

  • value The argument value to verify.
  • name string The name of the argument.
  • ... string The allowed types of the argument.
