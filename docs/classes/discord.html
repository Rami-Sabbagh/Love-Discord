Methods

discord:initialize(tokenType, token, connectInstantly, gatewayOptions)
Create a new instance of the Discörd library, and loads the submodules of the library.

Parameters:

  • tokenType string The OAuth authorization token type. Can be Bot or Bearer.
  • token string The OAuth authorization token.
  • connectInstantly optional boolean Whether to start the gateway connection automatically or not (false by default).
  • gatewayOptions optional table Set custom options for the gateway.

See also:

    discord:connect
discord:update(dt)
Update the gateway and process gateway events.

Parameters:

  • dt number The time between the last update call and this call in seconds.

Events system methods

Methods for using the events system of the library.
discord:hookEvent(name, func)
Hook a function into an event.

Parameters:

  • name string The event name.
  • func function The function you wish to hook.
