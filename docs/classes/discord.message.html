Class discord.message

The discord message class.

It can be obtained either from a discord event, or by requesting a message by it's ID from the discord servers.

Usage:

    local fetchedMessage = discord.message("channel_id", "message_id") --The requested message.

Methods

discord.message:initialize(data, messageID)
Create a new message object. Either from an internal message data table, or by requesting the message data from discord servers.

Parameters:

  • data table or string Either a message data table (from an internal api), or a Guild ID as a string.
  • messageID optional string The Message ID as a string to fetch, only when data is a Guild ID.

Raises:

Request error when fails to request the message from discord servers.
discord.message:addReaction(emoji)
Add a reaction to the message.

Parameters:

  • emoji string or discord.emoji The reaction to add, either a string which is the name of a standard emoji (ex sweat), or an emoji object.

Raises:

Request error on failure.
discord.message:delete()
Delete the message.

Request error on failure.
discord.message:edit(content, embed)
Edits the message.

Parameters:

  • content optional string The new message content, standard emojis are automatically replaced with their unicode character.
  • embed optional discord.embed The new embed content of the message.

Raises:

Messages content can't be longer than 2000 characters!, Either content or embed parameter has to be present! or request error on failure.

discord.message:getReplyChannel()
Get a fake channel object for ONLY replying to the message.

Only the channel ID has a proper value, and the channel type is just set into GUILD_TEXT for messages with guild ID , and into DM for other messages.

Other channel fields are just nil.

Returns:

    discord.channel A fake channel object to send a reply message using it.
